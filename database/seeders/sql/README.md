# سكربتات إدراج المدن واللوحات يدوياً

## الترتيب
1. **المدن أولاً:** نفّذ أحد ملفات المدن.
2. **اللوحات ثانياً:** نفّذ `02_plates_manual.sql`.

## ملفات المدن (اختر واحداً)

| الملف | متى تستخدمه |
|-------|-------------|
| `01_cities_manual.sql` | إذا كان على جدول `cities` مفتاح **UNIQUE** لـ `name_ar`. إذا لم يكن موجوداً أضفه: `ALTER TABLE cities ADD UNIQUE KEY unique_name_ar (name_ar);` ثم نفّذ الملف. |
| `01_cities_manual_no_unique.sql` | إذا لم ترد إضافة مفتاح فريد: يُدرج المدن الجديدة ويحدّث `name_en` و `order` للمدن الموجودة. |

## اللوحات
- **`02_plates_manual.sql`**  
  يُدرج لوحة افتراضية واحدة لكل مدينة (برقم = رقم اللوحة/المنطقة) إذا لم توجد، ويحدّث `plate_number` للوحات الموجودة ليطابق `order` المدينة.

## طريقة التنفيذ (MySQL/MariaDB)

من سطر الأوامر:
```bash
mysql -u المستخدم -p اسم_قاعدة_البيانات < 01_cities_manual_no_unique.sql
mysql -u المستخدم -p اسم_قاعدة_البيانات < 02_plates_manual.sql
```

أو من عميل SQL (HeidiSQL, phpMyAdmin, DBeaver...): افتح الملف ونفّذ المحتوى.
